## 📌 Project Overview

Nagraj Industries HRMS is a comprehensive Human Resource Management System developed during my internship to digitize and simplify workforce operations within an industrial environment.

The application enables seamless management of employees, contractors, departments, and organizational activities through dedicated role-based dashboards. Different roles such as CEO, HR, Production Head, Contractor, and Admin have access to features tailored to their responsibilities, ensuring efficient workforce management and operational transparency.
